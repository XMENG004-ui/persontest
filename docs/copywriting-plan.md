# 文案与商品化调整清单

## 1. 品牌语气基调

- **定位**：轻量付费心理探索，兼具神秘感与可靠度。  
- **关键词**：黑暗指数 / 情绪防御 / 情绪价值 / 匿名测试 / 快速出报告。  
- **语气**：冷静、克制、略带戏剧性；避免过度学术或夸张诊断用语。  
- **核心提示**：始终附带“仅供娱乐与自我探索参考”。

## 2. 页面级文案要点

| 页面/模块 | 文案需求 | 说明 |
| --- | --- | --- |
| **封面页** | 标题：“测一测你的潜在黑暗人格指数”；副副标题强调“匿名 · 3-12 分钟 · 即刻出报告”；CTA：“开始探索我的暗面”。 | 保留氛围文字，但增加“支付后领取专属链接”提醒。 |
| **卖点条** | 三条 bullet：① 36/96 题精细剖析 ② 匿名作答+长图报告 ③ 不构成诊断。 | 靠近 CTA，增强购买信心。 |
| **版本卡** | 标题＋副标题（基础=快速预判 / 深度=全景剖析）；价格文案（例如“¥1.99 / ¥2.99”，可根据定价修改）；耗时和报告差异；“推荐”徽标仅贴在深度版。 | 另加一句“含全部六维指数 + 原型匹配”。 |
| **免责声明勾选** | 文案：“我已了解本测试为娱乐体验，不构成专业心理诊断或治疗建议。” | 必须用户勾选后才能开始。 |
| **作答提示** | 顶部提示：“请选择最贴近你真实常态的选项；无需完美答案。”；底部“作答中断可稍后继续”。 | 减少纠纷。 |
| **进度状态** | 文案：“第 12 / 36 题 · 约 2 分钟可完成”。 | 动态填充题号与预估剩余时间。 |
| **提交 Loading** | “正在解析你的黑暗指数 · 预计 3 秒”。 | 给用户预期。 |
| **结果页头部** | 主标题：“你的潜在黑暗人格指数：72”；副标题：“数据基于 96 道题 · Latent Dark Index v1.0”；引用句（来自荣格等）放在次级位置。 | 强化可信度。 |
| **雷达说明** | “六维策略雷达：越接近外圈代表该策略越常被调用。” | 用户教育。 |
| **维度卡片** | 结构：得分+级别+一句总结+一句建议；结尾加提示“仅供娱乐”。 | 引导截图留文案。 |
| **原型模块** | 标题：“你的主导策略：高防御型控制者”；副标题：“以精密布局换取安全感”；结尾 CTA：“查看提升建议”。 | 语言需更“商品化”，强调价值。 |
| **亮点/风险** | 亮点：“擅长在混沌中迅速接管局面”；风险：“过度情绪操作可能稀释信任”。 | 每条 ≤ 18 字方便排版。 |
| **分享 CTA** | 主 CTA：“保存报告长图”；次 CTA：“复制文案分享到小红书”；附带说明“禁止擅自商用”。 | 贴合用户使用习惯。 |
| **复购 Banner** | 文案：“还可以继续测：爱情占有欲 / 童年阴影 / 幸福体感”；按钮：“去店铺查看更多测试”。 | 引导二次转化。 |
| **客服入口** | 文案：“遇到问题？联系店主” + 微信/小红书号提示。 | 减少售后摩擦。 |

## 3. 付费说明与提醒

| 节点 | 文案 | 位置 |
| --- | --- | --- |
| 下单后消息 | “请在 48 小时内使用专属链接；每个链接可多次复测，支持截图保存。” | 店铺自动私信。 |
| 页面顶角 | “娱乐体验 · 不收集个人身份 · 如需专业帮助请咨询心理机构。” | 所有页面统一角标或页脚。 |
| 提交前弹窗 | “确认提交并生成报告？提交后可在结果页反复查看。” | 最后一题点击提交时。 |
| 结果页底部 | “©2026 LATENT DARK INDEX · 数据仅供娱乐参考，不构成医学/心理诊断。” | 页脚。 |

## 4. 文案资产交付建议

- `copy/landing.json`：包含首屏标题、副标题、卖点、CTA。  
- `copy/version.json`：基础/深度版文案、价格、推荐标签。  
- `copy/question.json`：提示语、按钮文案、提醒。  
- `copy/result.json`：总分标题模板、雷达说明、分享 CTA、复购文案。  
- `copy/disclaimer.json`：与 `data/disclaimers.json` 对齐，用于多处引用。
